## How to use
1. Unzip the `Ashirwad_Buildcon_frontend.zip`.
2. Open `index.html` in a browser (or push to GitHub and use GitHub Pages).
3. For PWA install, serve the site over HTTPS or localhost and the browser will prompt install.

**Note:** This is a frontend-only project. Forms and AI tools are demo UI only and do not connect to any backend.
